# Mini Code Lab

Bienvenidos a **Mini Code Lab**, punto de encuentro para desarrolladores. Hemos creado Mini Code como plataforma de colaboración entre desarrolladores, desde los que estáis dando los primeros pasos hasta los que habéis vivido la evolución del desarrollo web.

Mejor que soltaros un discurso super molón vamos a responder las preguntas que nos hicimos antes de lanzar **Mini Code Lab**.

### ¿Cuál es el problema? 🤷🏽

Desarrolladores con mucho que compartir pero sin tiempo para difundir su conocimiento en diversas redes sociales o plataformas como Twitch o Youtube. "¡Estaría genial que existiese un altavoz!" pues de ahí surge Mini Code Lab.

### ¿Cómo va a funcionar Mini Code? 🧪

Tendremos mínimo un directo semanal vía **Twitch**. Esperamos próximamente anunciar colaboraciones de amigos y compañeros desarrolladores. Todo lo que resulte interesante para un desarrollador tendrá cabida en Mini Code. También postearemos en nuestra web, crearemos contenido en git y un poco de alquimia dev en las redes sociales 🪄.

### ¿Vamos a tener talleres?

Por supuesto, tendremos talleres de iniciación y profundización en diversas tecnologías y frameworks. Desde los básicos de Javascript hasta NextJS... todo ello acompañado de documentación y diversos repositorios que os sirvan de apoyo en el aprendizaje.

### ¿Tenemos guías dev? 🧑🏻‍💻

Sí, os explicaremos los caminos que debería seguir un desarrollador y tecnologías que debe dominar para alcanzar la meta propuesta.

### ¿Cúando conoceremos a más desarrolladores?

Vamos a traer al canal a nuestros amigos tech que viven diferentes momentos en su carrera para que los conozcáis y veáis cómo ha sido su evolución.

### ¿Quiénes iniciamos el proyecto?

Somos un grupo de desarrolladores con muchas ganas de devolver a la comunidad todo lo que han recibido.
