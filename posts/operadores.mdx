# **Guía Mini Code de Javascript - Operadores lógicos & aritméticos**

Los operadores lógicos comparan valores booleanos y devuelven respuestas booleanas. Vamos a ver alguno de ellos.

El operador OR, representado por **`||`**, devuelve verdadero si uno de los valores combinados es verdadero.

```jsx
let isAhero = false;
let isMarvelHero = true;
let puedoSerAvenger = isAhero || isMarvelHero;
// Devuelve true, porque es un super heroe de Marvel
```

El operador AND, representado por **`&&`**, devuelve verdadero solo si todos los valores combinados son verdaderos.

```jsx
let isAhero = false;
let isMarvelHero = true;
let puedoSerAvenger = isAhero && isMarvelHero;
// Devuelve false, porque no es un super heroe
```

Por último, pero no menos importante, tenemos el operador NOT de negación **`!`**. Se usa para negar el valor de una expresión (darle el valor opuesto).

```jsx
!true; // => false
!false; // => true
```

Los operadores aritméticos en Javascript nos permite realizar tantas operaciones matemáticas como queramos. Os dejamos algunos ejemplos de operaciones en Javasacript.

```jsx
let numa = 5;
let numb = 2;
numa + numb; // 7
numa - numb; // 3
numa * numb; // 10
numa / numb; // 2.5

let num1 = ++numa; // 6
let num2 = --numb; // 1
```
