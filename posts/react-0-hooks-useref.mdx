Vamos a crear un componente muy sencillo que tenga un input y lo enlazaremos a una referencia e imprimiremos el valor. Esto provocacar√° un nuevo renderizado solo cuando invoquemos a `printValue` en lugar de hacerlo a cada cambio del input.

  );
